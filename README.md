# covid-icu
Predicción temprana de mortalidad intrahospitalaria en pacientes adultos ingresados en UCI con Covid-19 mediante modelos de **Machine Learning**, a partir de datos clínicos y fisiológicos.

## Descripción

El proyecto consiste en desarrollar y evaluar modelos de Machine Learning para predecir de forma temprana la mortalidad intrahospitalaria en pacientes adultos ingresados en UCI con Covid-19.  Se trabajará con datos clínicos reales, realizando un proceso completo desde la adquisición y preprocesamiento hasta la evaluación y posible interpretación de los modelos.

## Objetivos

- Acceder a los datos necesarios completando los cursos o pasos previos requeridos por la plataforma fuente.
- Realizar el preprocesamiento y limpieza de los datos, incluyendo tratamiento de valores perdidos y estandarización de variables.
- (Opcional) Aplicar técnicas de ingeniería de características para extraer predictores clínicamente relevantes.
- Desarrollar y evaluar modelos de Machine Learning para la predicción temprana de la mortalidad en UCI.
- (Opcional) Utilizar técnicas de interpretabilidad para identificar los factores clínicos y fisiológicos más influyentes en la predicción.

## Tecnologías

- Manipulación de datos: Pandas y PySpark.
- Machine Learning: Scikit-learn.
- Deep Learning: TensorFlow/Keras o PyTorch, Hugging Face Transformers.
- Visualización: Matplotlib, Seaborn, WordCloud, Plotly (opcional).
- Entorno de desarrollo: Jupyter Notebooks, VS Code, Google Colab (para GPUs si se usa Deep Learning).
- Control de versiones: GitHub
